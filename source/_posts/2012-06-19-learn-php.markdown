PHP基本语法：   

四种标量类型：   

*    boolean （布尔型）
*    integer （整型）
*    float （浮点型, 也称作 double)
*    string （字符串） 

两种复合类型：

*    array （数组）
*    object （对象） 

最后是两种特殊类型：

*    resource　（资源）
*    NULL　（NULL） 

为了确保代码的易读性，本手册还介绍了一些伪类型：

*    mixed
*    number
*    callback
